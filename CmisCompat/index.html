Test procedure

  1. Download the latest Apache Chemistry CMIS Workbench
  2. Run CMIS Workbench and log into your ECM server. If given a choice, choose the repository that typically contains most project documents
  3. Press the TCK button, let every option at their default value, and press run
  4. The tests will run for a few minutes, and at the end a results dialog will show up
  5. Select HTML, a browser window will open showing the details
  6. Press CTRL-S to save the HTML
  7. Send the HTML file to our email address or send a pull request to our Github. Don't forget to send us the server version, the version of CMIS Workbench, and the CMIS URL address used.

Contribute your results

We don't have access to all ECM products. Please help us by testing the products you have access to. The test procedure is detailed in the paragraph above. Thanks!

This page is a collaborative effort, hosted on Github. Don't hesitate to join us and send us pull requests. Our goal is to improve ECM interoperability by forcing all ECM vendors to make sure their CMIS implementation is correct.
