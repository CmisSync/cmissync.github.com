CmisSync

Dropbox-like sync for Alfresco!

CmisSync allows you to keep in sync with your company's Alfresco, and read/edit the documents even when offline:

Step 1

Install .NET, then CmisSync.

Step 2

Run CmisSync.
After a few welcome screens, it will start running as an icon in the tray:

Click on the tray icon and select "Add a CMIS folder".

Step 3

A dialog appears. In the address field, copy-paste the URL of your Alfresco server, it is the web address you use to access Alfresco in your web browser:

Insert the https:// prefix if it did not get copied.
Then enter your Alfresco username/password.

Step 4

The next screen shows the Alfresco folders.
Choose the folder you want to sync.
If you use Alfresco Share (like most companies), click "Sites", select your site, then click "documentLibrary".

That's all!
CmisSync will now sync Alfresco to your desktop, whenever you have a network connection.

CmisSync does two-way sync, which means you can modify/add files.
Should a conflict occur, a backup is made and you can handle it by choosing which version to keep.

While syncing is also possible with Alfresco Desktop Sync, the CmisSync client has some advantages:
  • Sync any Alfresco content, not just Alfresco Share Sites
  • Based on CMIS, modern and more network-friendly than WebDAV
  • Frequent releases based on users feedback
  • Open Source (volunteers welcome!)
